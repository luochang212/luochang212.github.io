前段时间朋友买重疾险，销售给的方案是带返还的，返还也算一种理财玩法吧。

这是销售给的方案：

  • 费用：分 30 年支付，每年支付 9000 元，共计 27 万
  • 权益：终生享受重疾险；身故后保险公司一次性返还 50 万

乍一看这个方案很划算，只要支付 27万，身故时能得 50万，数字近乎翻倍，重疾险就像白送的。但考虑到 30 年起步的资金成本，还是要具体计算一下才知道值不值。

一、折现率和净现值NPV

虽然“保险价值”人各不同，但“理财价值”却是可以客观衡量的。

让我们把理财部分单独剥离出来分析，此时它的损益如下：

0 1 2 3 … 29 30+（以身故时间为准）
-9,000 -9,000 -9,000 -9,000 … -9,000 +500,000

我们知道，资金有时间成本。来自不同时期的钱，其价值是不同的。为了计算真实的损益情况，我们引入 折现率 和 净现值 两个概念。

1.1 什么是折现率？

折现率 (Discount rate) 是把未来的钱折算成现在的钱的折扣率。如果你认为一年后的 107 元与现在 100 元相当，折现率就是 7%，用公式表达为：

$一年后107元的现值 = \frac{107}{1 + 折现率} = \frac{107}{1 + 7\%} = 100$

1.2 什么是净现值？

净现值(NPV) (Net present value) 指未来现金流入现值与未来现金流出现值的差额。

一旦有了上面那张损益表，并有了一个确定的折扣率，就能计算 NPV。假设折扣率为 0.02，有：

$NPV = -9000 -9000*\frac{1}{1 + 0.02} -9000*\frac{1}{(1 + 0.02)^{2}} + ... + -9000*\frac{1}{(1 + 0.02)^{29}} + 500000*\frac{1}{(1 + 0.02)^{t}}$

其中 t 为从交保到身故的时间间隔，单位为年，值域 $[30, +\infty]$

根据 NPV 公式，可写如下 Python 函数。后面会多次用到它，可以看一眼。

# 计算成本、收益、NPV
def calc_npv(discount_rate=0.02, t=30):
    """返回 成本的现值、收益的现值、净现值NPV
    """
    cost = sum([9000/((1+discount_rate)**i) for i in range(30)])
    profit = 500000/((1+discount_rate)**t)
    return cost, profit, profit - cost

二、成本-收益曲线

由于 NPV = 收益的现值 - 成本的现值，所以展示 成本和收益 而非 NPV，让我们更容易理解公式内部发生了什么。

下图是用偏保守的 2% 折现率计算的 成本-收益曲线。

图中，横坐标是身故时间（以买保险的时刻为 0，单位年），纵坐标是现值（单位元）。图中的两条线分别表示成本现值和收益现值的走势。

由于成本 cost 只与折扣率有关，与时间无关，因此是一条横线；收益 profit 随时间递减，这导致 NPV 在交保后的第 45 年转负 (NPV = profit - cost)，这代表第 45 年之后就会亏损。

注意，上述分析只考虑理财价值，不考虑保险价值。因此 NPV 转负的实际时间要更晚。但也别忘了，我们刚刚用的是 2% 的保守折现率，如果用更接近现实世界的数字，会发生什么呢？

画图代码：

# 成本/收益曲线
import matplotlib.pyplot as plt

discount_rate = 0.02
npv = [calc_npv(discount_rate, t) for t in range(30, 60)]
x_list = list(range(30, 60))
cost_list = [e[0] for e in npv]
profit_list = [e[1] for e in npv]

plt.plot(x_list, cost_list, label='cost')
plt.plot(x_list, profit_list, label='profit', marker='.')
plt.title('cost-profit curves')
plt.xlabel('(years)')
plt.ylabel('(yuan)')
plt.legend()
plt.show()

Note: 2% 的折现率保守吗？

你可以参考一下这些 2023 年的信息：

  • 中国青年报：过去十年CPI涨幅是 2%
  • 货币基金的年化收益率一般是 2%，大额存单可达 3%
  • 30 年将跨越好几个经济周期，如遇通胀周期，2% 跑不赢通胀

三、不同折现率的NPV曲线

由于 2% 的折现率太低，我们看看 3%, 5%, 7% 下 NPV 的表现。

从 NPV 转负的时间看。3% 的折现率，缴满 5 年后转负；7% 的折现率，缴满之后就已经是负的了，而且是 -5.3 万。

从预期寿命下的 NPV 看。假设预期寿命是 80 岁，此时 t 近似取 50。7% 折现率对应 80 岁时亏 10.2 万；3% 折现率对应亏 6.7 万。

🦈 简单做个结论：

因为不知道更细致的分布情况，无法做更深入的计算。所以粗略的结论是，如果你愿意花 6-10 万的现值，换终身重疾险，并留 50 万的税前遗产给受益人，那这款产品可能适合你。这么算下来，这款产品过分良心了，我对 50 万是不是无条件返还表示强烈怀疑。

画图代码：

# 不同折现率，对应的净现值曲线
x_list = list(range(30, 60))
get_npv_list = lambda e: [calc_npv(e, t)[2] for t in range(30, 60)]
dr_02 = get_npv_list(0.02)

plt.plot(x_list, dr_03, label='3%')

plt.title('NPV curves')

Note: 7% 折现率参考值的依据

折现率的具体值，一般考虑通胀、资金成本、延迟消费等因素。理论上，一笔用于投资的钱，最好的年化收益率是 5%，它的折现率就是 5%

7% 是 OMB 提供的参考值，维基百科 原文如下：

“The US government uses a variety of discount rates but something around 7% is what the US Office of Management and Budget (OMB) recommends for a pretax rate of return on private investments. In the United Kingdom, HM Treasury fixes the social discount rate for the public sector at 3.5%.”

四、内部收益率IRR

IRR 可以作为购买保险的参考指标，IRR 越高对顾客越有利。

内部收益率 (IRR) 是净现值（NPV）等于 0 时的折现率。将该定义结合 NPV 公式，有如下代码：

import numpy as np
from scipy.optimize import fsolve

def calc_root(t=50):
    f = lambda e: 500000/((1+e)**t) - sum([9000/((1+e)**i) for i in range(30)])
    return fsolve(f, 0)[0]

root = calc_root()
print(root)

运行上述代码，输出：$root \approx 0.017$

这意味着，80 岁时身故（对应参数 t=50）对应的内部收益率 IRR 约为 0.017。PS: 不要忘了 0.017 是只看保险理财属性的结果。

五、我的看法

“保险值不值得买” 不是一个技术问题，它是现实问题。小到保险公司乱收费，大到通胀周期发生，都不是几个简单指标可以预测的，真正做决策购买，还需要综合考虑更多现实因素。

就我个人而言，不是很喜欢理财型保险。可能是我的偏见。下面是我觉得它存在的风险，供你参考：

  • 履约成本：30年长周期，叠加高年费，有断供风险
  • 通胀周期：30年会经历多轮经济周期。如遇通胀，像 2023 年俄罗斯那样就亏大了
  • 资金流动性：资金水位较低时，可能影响资金流动性，富哥富姐无视
  • 资金机会成本：保费锁死了一部分钱，隐形中你失去了其他投资机会
  • 收益上限：保险理财提供的收益高吗？如果不高，为什么这么多钱要被它锁住？
  • 收益兑付：手续麻烦吗？快吗？受益人要不要纳税？
  • 保险公司倒闭的风险：你确定保险公司活得比你久？
  • 医保的进步：医保变好，商业保险就变鸡肋了
