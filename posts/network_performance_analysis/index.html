排队论在网络性能分析中的应用

Jul 4, 2019 in MATH • QUEUING THEORY
network performance analysis queuing theory
2 min read
      • 柏松过程的推导

排队论(queueing theory)也称随机服务系统理论，就是为解决上述问题而发展的一门学科，它研究的内容有三部分：性态问题、最优化问题、统计推断问题。（《运筹学》清华大学出版社）

下图是书中用于解释排队过程的流程图。

现实中的排队问题是多种多样的，对上述“顾客源”和“服务机构”应该作宽泛的理解。顾客和服务机构可以是生物，也可以是非生物；排队结构可以是有形的，也可以是无形的，比如向交换台要求通话的请求；顾客可以走向服务机构，也可以相反，比如送货上门。

本文着重介绍网络性能分析(Network performance analysis)中常见的几种排队模型。它们是：

  • The M/M/1 queuing system
  • The birth-death process
  • The M/M/m queuing system
  • The M/M/1/m queuing system
  • The M/M/∞ queuing system
  • The M/M/m/m queuing system

另外，作为以上模型的理论基础，本文将首先介绍柏松分布(The Poisson distribution)。

柏松过程的推导

柏松过程是马尔可夫过程的一个特例，在排队论中经常被用来估计顾客到达的概率分布。

（一）柏松过程的假设

  • 在一段时间间隔$\Delta t$内，有且仅有一位顾客到达的概率$P$和$\Delta t$成比例，比例系数为$\lambda$

  • 在一段时间间隔$\Delta t$内，至多只允许一位顾客到达（否则应该选取一个更短的时间间隔） $$P(在时间间隔[t, t + \Delta t]内，有且仅有一位顾客到达) = \lambda \Delta t \\P(在时间间隔[t, t + \Delta t]内，没有顾客到达) = 1 - \lambda \Delta t \\P(在时间间隔[t, t + \Delta t]内，多于一位顾客到达) = 0$$

  • 柏松过程是对顾客到达过程的估计，故不考虑顾客的离开情况，所以对到达人数的计数只增不减

  • 只考虑一阶无穷小对结果的影响，忽略二阶无穷小，$\Delta t \gg(\Delta t)^{2} \approx 0$

  • $P_{k}(t)$表示在时间间隔$t$内有k位顾客到达的概率

  • $p_{i, j}(\Delta t)$表示在时间间隔$\Delta t$内到达总人数由$i$变为$j$的概率

（二）推导过程

由于总人数单调递增，且在一段时间间隔$\Delta t$内，到达人数最多加1。因此，如果$t+\Delta t$时刻的到达人数是k，那么在$t$时刻的到达人数只有两种情况：k或者k-1.

从时间$t$到时间$t+\Delta t$，到达人数的概率可由下述公式表达： $$\begin{aligned} P_{k}(t+\Delta t) &=P_{k}(t) p_{k, k}(\Delta t)+P_{k-1}(t) p_{k-1, k}(\Delta t) \\ P_{0}(t+\Delta t) &=P_{0}(t) p_{0,0}(\Delta t) \quad k=0 \text { (starting condition) } \end{aligned}$$

由假设可知： $$p_{k, k}(\Delta t)=(1-\lambda \Delta t) \quad \text { and } \quad p_{k-1, k}(\Delta t)=\lambda \Delta t$$

将此式代入上式可得： $$\begin{aligned} P_{k}(t+\Delta t) &=P_{k}(t)(1-\lambda \Delta t)+P_{k-1}(t)(\lambda \Delta t) \\ P_{0}(t+\Delta t) &=P_{0}(t)(1-\lambda \Delta t) \quad k=0 \text { (starting condition) } \end{aligned}$$

整理得： $$\begin{aligned} \frac{P_{k}(t+\Delta t)-P_{k}(t)}{\Delta t} &=-\lambda P_{k}(t)+\lambda P_{k-1}(t) \\ \frac{P_{0}(t+\Delta t)-P_{0}(t)}{\Delta t} &=-\lambda P_{0}(t) \end{aligned}$$

当$\Delta t \rightarrow 0$有： $$\begin{aligned} \frac{d P_{k}(t)}{d t} &=-\lambda P_{k}(t)+\lambda P_{k-1}(t) \quad (k \geq 1) \quad (1) \\ \frac{d P_{0}(t)}{d t} &=-\lambda P_{0}(t) \quad \quad \quad \quad (2) \end{aligned} $$

方程(2)的解为： $$P_{0}(t)=A \exp (-\lambda t)$$

其中$A$是任意常数。设$k=1$，将方程(2)的解代入方程(1)有： $$\frac{d P_{1}(t)}{d t}=-\lambda P_{1}(t)+A \lambda \exp (-\lambda t)$$

解得： $$P_{1}(t)=A \lambda t \exp (-\lambda t)$$

$k=2$时同理： $$\frac{d P_{2}(t)}{d t}=-\lambda P_{2}(t)+A \lambda^{2} t \exp (-\lambda t)$$

解得： $$P_{2}(t)=\frac{A(\lambda t)^{2}}{2} \exp (-\lambda t)$$

 对任意的k，解为： $$P_{k}(t)=\frac{A(\lambda t)^{k}}{k !} \exp (-\lambda t)$$

下面计算$A$的值，由于： $$P_{0}(t)+P_{1}(t)+P_{2}(t)+\cdots=\sum_{i=0}^{\infty} P_{i}(t)=1$$

故有： $$A \exp (-\lambda t) \sum_{k=0}^{\infty} \frac{(\lambda t)^{k}}{k !}=1$$

由泰勒展开可知$A=1$。故柏松过程的一般式为：

$$P(k | t, \lambda)=\frac{(\lambda t)^{k}}{k !} \exp (-\lambda t)$$
