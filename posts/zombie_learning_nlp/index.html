PyTorch 环境准备

安装 PyTorch：MacOS，使用Anaconda安装

conda install pytorch torchvision -c pytorch

其他系统的安装方法见：官网安装教程

安好后，在命令行简单验证一下

python -c "import torch;x = torch.rand(5, 3);print(x)"

查看版本号

python -c "import torch;print(torch.__version__)"
