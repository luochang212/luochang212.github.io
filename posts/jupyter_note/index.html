Jupyter Lab 你是真的火了

5 月 14 号 OpenAI 春季发布会 上出现这样一张图：

让我们知道，OpenAI 的研究员也在使用 Jupyter Lab

正好，我最近正在从 Jupyter Notebook 转向使用 Jupyter Lab 做开发。Jupyter Lab 其实有丰富的功能，并不只是 Python 的解释器这么简单。看看我都挖掘了哪些功能吧！

一、制作 PPT

1.1 Notebook 转 PPT

只需两步，就能把 Notebook 变成 PPT：

  1. 设置 Slide Type：你需要对每个 Cell 设置 Slide Type，才能保证页面被正确渲染。在新版 Jupyter Lab 的右侧边栏上，有一个带俩齿轮的按钮。点进去，然后一路点 [Property Inspector] -> [COMMON TOOLS] -> [Slide Type]，对 Cell 进行设置。

  2. 打开网页应用：打开 Terminal，来到 [YOUR_SLIDE].ipynb 文件所在路径，运行

    jupyter nbconvert --to slides [YOUR_SLIDE].ipynb --post serve

    这会在浏览器中打开一个网页应用，你可以用这个网页做 pre。值得一提的是，这个网页是有路由的，这意味着如果你在服务器上运行 Jupyter，你甚至可以通过网页链接分享特定页面。

1.2 把 PPT 存成 PDF

你可以把 PPT 存成 PDF，只需要：

  1. 把网页路由的 # 号改成 ?print-pdf
  2. 用 Chrome 的打印功能，把网页存成本地 PDF

修改后的网址长这样：

http://127.0.0.1:8000/[YOUR_SLIDE].slides.html?print-pdf

二、在 Jupyter 中运行 Java

IJava 支持在 Jupyter 环境中嵌入 Java 内核。它允许用户在 Java 编程中使用 IPython 的特性，如交互式编程、魔术命令、图表输出等。

2.1 安装 IJava

  1. 从 Releases 下载安装包，安装包形如 ijava-[VERSION].zip
  2. 解压 ijava-[VERSION].zip，进入文件夹中运行 
    python3 install.py --sys-prefix
  3. 检查内核是否安装成功 
    # 列出已安装的内核
jupyter kernelspec list

# 尝试在命令行中运行 java 内核
jupyter console --kernel=java

2.2 内核管理工具

jupyter kernelspec 相关的命令选项可用于安装、管理、移除内核，以下是一些常用命令：

  • 列出内核：jupyter kernelspec list
  • 安装内核：jupyter kernelspec install [KERNEL_DIRECTORY]
  • 移除内核：jupyter kernelspec uninstall [KERNEL_NAME]

Note: 使用 jupyter kernelspec install 命令安装内核时，可指定多项参数，下面是一个示例命令：

jupyter kernelspec install \
  ~/miniforge3/share/jupyter/kernels/java \
  --sys-prefix \
  --name java \
  --display-name "Java"

在这个命令中：

  • ~/miniforge3/share/jupyter/kernels/java 是内核路径，可用 jupyter kernelspec list 命令查看内核路径
  • --sys-prefix 选项确保内核在系统范围内可用，而不仅限于特定用户或虚拟环境。这对多用户系统或者需要在全局范围内共享内核的情况非常有用。如果你希望内核仅对当前用户可见，也可以使用 --user 选项
  • --name java 指定了内核的名 (kernel name)，这是在编程环境引用内核时使用的名称
  • --display-name "Java" 则设置了在 Jupyter Notebook 或 Jupyter Lab 界面中显示给用户的名称

三、Mermaid 插件

Mermaid 是一个流行的图表生成工具。安装 Mermaid 插件后，用户可以在 Markdown 单元格中使用 Mermaid 语法编写图表代码，并将其渲染成图表。Mermaid 支持多种图表类型，如流程图、序列图、甘特图等。

用以下命令安装 mermaid 插件：

jupyter labextension install @jupyter-widgets/jupyterlab-mermaid

四、变量查看器

jupyterlab-contrib/jupyterlab-variableInspector 是一个 Jupyter Lab 扩展，它允许用户直接查看交互中的变量。

用以下命令安装变量查看器：

pip install lckr_jupyterlab_variableinspector

安装后，在 Notebook 的空白处，通过右键菜单栏的 Open Variable Inspector 打开。

Note: 使用 Mamba

Mamba 是 Miniforge 自带的包管理工具。大多数时候，你可以用 mamba 命令直接替代 conda 命令，如 mamba install numpy -c conda-forge。一般来说，使用 mamba 能在安装时获得更高的下载速度。
