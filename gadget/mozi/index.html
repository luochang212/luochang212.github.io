介绍

  • 作者 : 墨翟
  • 成书时间 : [春秋 - 战国] 公元前490年-公元前221年
  • 篇目数量 : 原有71篇，现存53篇

近代中国，在西方的坚船利炮下被迫开放国门，西方思想也随之流入中国。在外来文化的冲击下，中国人开始重新审视自己的祖先。毫不意外地，酷肖西方的民主与科学精神的墨家思想被重新发现。

墨家主张“兼爱”，和西方的博爱相似。墨家还崇尚科学，这也与西方的科学精神源流不谋而合。当然，中国历史并却没有选择墨家，而是选择了更适合农业社会发展的儒家。这也许是一种必然，中国的地理气候以及人民的生活方式，决定了历史偏爱儒家。但墨家依旧代表了一种文明演化的可能性。从《墨子》这本书里，我们可以窥见2000多年前的祖先，面对当时的社会问题，是如何思考的。

看《墨子》，其实是在看我们的文化母体，在寻我们的根。

卷一 #back to top

亲士

  1. 入国而不存其士，则亡国矣。见贤而不急，则缓其君矣。非贤无急，非士无与虑国，缓贤忘士而能以其国存者，未曾有也。

    不急：不急于任用。缓：怠慢

  2. 昔者文公出走而正天下，桓公去国而霸诸侯，越王句践遇吴王之丑，而尚摄中国之贤君。三子之能达名成功于天下也，皆于其国抑而大丑也。太上无败，其次败而有以成，此之谓用民。

  3. 吾闻之曰：“非无安居也，我无安心也。非无足财也，我无足心也。”是故君子自难而易彼，众人自易而难彼，君子进不败其志，内究其情，虽杂庸民，终无怨心，彼有自信者也。是故为其所难者，必得其所欲焉，未闻为其所欲，而免其所恶者也。是故逼臣伤君，谄下伤上。君必有弗弗之臣，上必有詻詻之下。分议者延延，而支苟者詻詻，焉可以长生保国。

    上必有詻詻（è）之下：上位者必须有直言极谏的下属。分议者延延，而支苟者詻詻：分辩议事的人争论锋起，互相责难的人互不退让。

  4. 臣下重其爵位而不言，近臣则喑，远臣则唫，怨结于民心，谄谀在侧，善议障塞，则国危矣。桀纣不以其无天下之士邪？杀其身而丧天下。故曰：“归国宝，不若献贤而进士。”

    喑（yīn）：沉默不语。唫（jìn）： 假借为“噤”，口闭。谄谀（chǎn yú）。

  5. 今有五锥，此其銛，銛者必先挫。有五刀，此其错，错者必先靡，是以甘井近竭，招木近伐，灵龟近灼，神蛇近暴。是故比干之殪，其抗也；孟贲之杀，其勇也；西施之沈，其美也；吴起之裂，其事也。故彼人者，寡不死其所长，故曰：“太盛难守也。”

    銛（xiān）：锋利、锐利。错：磨。殪（yì）：沈：死。同“沉”。

  6. 故虽有贤君，不爱无功之臣；虽有慈父，不爱无益之子。是故不胜其任而处其位，非此位之人也；不胜其爵而处其禄，非此禄之主也。良弓难张，然可以及高入深；良马难乘，然可以任重致远；良才难令，然可以致君见尊。是故江河不恶小谷之满己也，故能大。圣人者，事无辞也，物无违也，故能为天下器。是故江河之水，非一源之水也。千镒之裘，非一狐之白也。夫恶有同方取不取同而已者乎？盖非兼王之道也。是故天地不昭昭，大水不潦潦，大火不燎燎，王德不尧尧者，乃千人之长也。

    镒（yì）：本义:古代的重量单位，二十两或二十四两为一镒。恶有：哪里有。是故天地不昭昭，大水不潦潦，大火不燎燎，王德不尧尧者，乃千人之长也：所以大地不昭昭为明（而美丑皆收），大水不潦潦为大（而川泽皆纳），大火不燎燎为盛（而草木皆容），王德不尧尧为高（而贵贱皆亲），才能做千万人的首领。

  7. 其直如矢，其平如砥，不足以覆万物，是故溪陕者速涸，逝浅者速竭，墝埆者其地不育。王者淳泽不出宫中，则不能流国矣。

    砥（dǐ）：本义:质地很细的磨刀石。逝浅者速竭：平浅的川泽枯得早。墝埆（qiào què）者其地不育：坚薄的土地不长五谷。

  • 笔记 : 列亲士于卷首，本身就说明了“尚贤”这个理念在墨家思想中的重要性。

    (1)中的“非贤无急，非士无与虑国”，说明了招纳贤才的紧迫性。治理社会需要贤达，有了他们的治理，社会才会趋于稳定，国家才能长保太平。

    (3)有点理解困难，因为文中出现“是故”的地方，逻辑跳跃很大。相连的句子间，也不是因果关系，而是一种需要理解的递进关系。第一句谈君子的志向，第二句谈君子的自信。但最后一句，居然开始说朝廷中要有敢于谏诤之人，国家才能蒸蒸日上。为什么逻辑跳跃会这么大呢？这中间逻辑的搭建也一度让我疑惑。但后来我想通了，应该是这么个意思：这段话想让君主理解君子是怎么想的，所以陈述了君子的志向以及君子的自信，这导出了君子会直言上谏的原因。因为君子依从自己的内心说话啊，不像佞臣会投君主所好，所以有时候会说出一些虽然是对的，但会引起君主反感的话。文中“是故为其所难者，必得其所欲焉，未闻为其所欲，而免其所恶者也”这句话，是理解此段的关键。它翻译过来是讲，凡事从难处做起，往往能得到自己想要的结果，而只做自己想做的事，则难免引发灾难性的后果。也就直接地规劝君王，虽然忠言逆耳，但还是非听不可。

    (5)有点意思，一直以为西施最终是和范蠡泛舟游湖去了，倒没听过沉湖一说。

    (6)开头第一句话疯狂示意，是说上位者如果不胜其任而处其位，那么就非此位之人。但是面对君王又怂，不敢明说，只能指桑骂槐，拿臣子和孩子说事。那么上位者需要具备哪些素质，才能胜其任处其位呢？关键就是要“兼容并蓄，百花齐放”。在上位者要足够大气，面对傲娇但能担大事的文士，能费心拉拢。面对人民，要雅俗共赏，不能因为俗就嫌弃。总而言之，就是要秉持着实事求是的精神，黑猫白猫，能捉老鼠的就是好猫。

修身

  1. 君子战虽有陈，而勇为本焉。丧虽有礼，而哀为本焉。士虽有学，而行为本焉。是故置本不安者，无务丰末。近者不亲，无务来远。亲戚不附，无务外交。事无终始，无务多业。举物而暗，无务博1闻。

    陈：战阵;行列。丰末：使末大于本。无务：不必讲究，不必追求。暗：昏昧，愚昧;不明白。

  2. 是故先王之治天下也，必察迩来远，君子察迩而迩修者也。见不修行，见毁，而反之身者也，此以怨省而行修矣。谮慝之言，无入之耳，批捍之声，无出之口，杀伤人之孩，无存之心，虽有诋讦之民，无所依矣。

    迩（ěr）：近。见不修行：看得出不修行。见毁：见，助词，表示被动。受人诋毁。谮慝（zèn tè）：诬蔑毁谤。批捍（pī hàn）：抨击。诋讦（dǐ jié）：诋毁攻击。

  3. 故君子力事日强，愿欲日逾，设壮日盛。君子之道也，贫则见廉，富则见义，生则见爱，死则见哀。四行者不可虚假，反之身者也。藏于心者，无以竭爱。动于身者，无以竭恭。出于口者，无以竭驯。畅之四支，接之肌肤，华发隳颠1，而犹弗舍者，其唯圣人乎！

    设壮日盛：庄敬的品行一天比一天完善。驯：顺从，温顺。隳颠（huī diān）：秃顶。

  4. 志不强者智不达，言不信者行不果。据财不能以分人者，不足与友。守道不笃，偏物不博，辩是非不察者，不足与游。本不固者末必几，雄而不修者，其后必惰，原浊者流不清，行不信者名必秏1。名不徒生而誉不自长，功成名遂，名誉不可虚假，反之身者也。务言而缓行，虽辩必不听。多力而伐功，虽劳必不图。慧者心辩而不繁说，多力而不伐功，此以名誉扬天下。言无务为多而务为智，无务为文而务为察。故彼智无察，在身而情，反其路者也。善无主于心者不留，行莫辩于身者不立。名不可简而成也，誉不可巧而立也，君子以身戴行者也。思利寻焉，忘名忽焉，可以为士于天下者，未尝有也。

    几（jī）：危险。雄而不修：光勇敢而不注重品行修养。无务为文而务为察：不图文采而讲究明白。

  • 笔记 : 本节谈修身，也就是品行修养。

    (1)差序格局大概深深根植于中华民族潜意识中。思考时，我们喜欢以己推人；谈论志向时，我们喜欢说“修身齐家治国平天下”。这些都是差序格局的体现。差序格局以自己为中心，像石头泛起的波纹一样，依照人际的亲疏远近，向外推去。本节中的“进者不亲，无务来远。亲戚不附，无务外交”也体现了这种思维习惯。

    (2)"君子察迩而迩修者也"和当今流行的生成对抗网络(GAN)颇有异曲同工之妙。“迩修”的过程，不正是生成对抗网络生成的过程吗，而“君子察迩”就是判别函数。

    为什么古代那么注重修身呢？大概是因为古代不存在现代的信用和安全机制，比如银行担保、芝麻信用、RSA加密。缺少这些机制的后果就是，信用与安全必须完全基于人，而人本身就是多变的动物，因此古圣人就必须教育士人修养品格、树立名誉，这样才能搭建社会信任的基础，才能高效地组织生产活动。

所染

  1. 子墨子言见染丝者而叹曰：“染于苍则苍，染于黄则黄。所入者变，其色亦变。五入必而已，则为五色矣。故染不可不慎也。”

    苍:深青色，深绿色。

  2. 非独染丝然也，国亦有染。舜染于许由1、伯阳，禹染于皋陶、伯益，汤染于伊尹、仲虺，武王染于太公、周公。此四王者所染当，故王天下，立为天子，功名蔽天地。举天下之仁义显人，必称此四王者。

    当：得当。仲虺（zhòng huǐ）：人名，商汤左相。

  3. 夏桀染于干辛、推哆，殷纣染于崇侯、恶来，厉王染于厉公长父、荣夷终，幽王染于傅公夷、蔡公谷。此四王者所染不当，故国残身死，为天下僇。举天下不义辱人，必称此四王者。

    僇（lù）：侮辱。

  4. 齐桓染于管仲、鲍叔，晋文染于舅犯、高偃，楚庄染于孙叔、沈尹，吴阖闾染于伍员、文义，越句践染于范蠡大夫种。此五君者1所染当，故霸诸侯，功名傅于后世。

  5. 范吉射染于长柳朔、王胜，中行寅染于籍秦、高强，吴夫差染于王孙雒、太宰嚭，知伯摇染于智国、张武，中山尚染于魏义、偃长，宋康染于唐鞅、佃1不礼。此六君者所染不当，故国家残亡，身为刑戮，宗庙破灭，绝无后类，君臣离散，民人流亡。举天下之贪暴苛扰者，必称此六君也。

    雒（luò）。嚭（pǐ）。偃（yǎn）。

  6. 凡君之所以安者，何也？以其行理也，行理性于染当。故善为君者，劳于论人，而佚于治官。不能为君者，伤形费神，愁心劳意，然国逾危，身逾辱。此六君者，非不重其国，爱其身也，以不知要故也。不知要者，所染不当也。

  7. 非独国有染也，士亦有染。其友皆好仁义，淳谨畏令，则家日益，身日安，名日荣，处官得其理矣，则段干木、禽子、傅说之徒是也。其友皆好矜奋，创作比周，则家日损，身日危，名日辱，处官失其理矣，则子西、易牙、竖刀1之徒是也。《诗》曰：“必择所堪。”必谨所堪者，此之谓也。

    矜奋：以勇气自恃;骄傲自大。创作：创建制作。比周：结党营私。

  • 笔记 : 所染篇以染布为喻，阐述了环境对人的影响。

法仪

  1. 子墨子曰：“天下从事者，不可以无法仪，无法仪而其事能成者无有也。虽至士之为将相者，皆有法，虽至百工从事者，亦皆有法。百工为方以矩，为圆以规，直衡以水，以绳，正以县。无巧工、不巧工，皆以此五者为法。巧者能中之，不巧者虽不能中，放依以从事，犹逾己。故百工从事，皆有法所度。”

    县（xuán）：古同“悬”。无巧工、不巧工：不论是巧匠还是一般工匠。

  2. 今大者治天下，其次治大国，而无法所度，此不若百工辩也。然则奚以为治法而可？当皆法其父母，奚若？天下之为父母者众，而仁者寡，若皆法其父母，此法不仁也。法不仁不可以为法。当皆法其学，奚若？天下之为学者众，而仁者寡，若皆法其学，此法不仁也。法不仁不可以为法。当皆法其君，奚若？天下之为君者众，而仁者寡，若皆法其君，此法不仁也。法不仁不可以为法。故父母、学、君三者，莫可以为治法。

    此不若百工辩也：这是不如工匠们能明辨事理。奚若：何如，如何，怎么样。

  3. 然则奚以为治法而可？故曰莫若法天。天之行广而无私，其施厚而不德，其明久而不衰，故圣王法之。既以天为法，动作有为，必度于天，天之所欲则为之，天所不欲则止。然而天何欲何恶者也？天必欲人之相爱相利，而不欲人之相恶相贼也。奚以知天之欲人之相爱相利，而不欲人之相恶相贼也？以其兼而爱之，兼而利之也。奚以知天兼而爱之，兼而利之也？以其兼而有之，兼而食之也。

    贼：害，伤害。

  4. 今天下无大小国，皆天之邑也。人无幼长贵贱，皆天之臣也。此以莫不犓羊牛、豢犬猪，洁为酒醴粢盛，以敬事天，此不为兼而有之，兼而食之邪？天苟兼而有食之，夫奚说以不欲人之相爱相利也？故曰：“爱人利人者，天必福之，恶人贼人者，天必祸之。”曰：“杀不辜者，得不祥焉。夫奚说人为其相杀而天与祸乎？是以知天欲人相爱相利，而不欲人相恶相贼也。”

    邑（yì）：城市，都城。犓（chú）：用草料喂牲畜。豢（huàn）：喂养。醴（lǐ）：甜酒。粢（zī）：谷子，子实去壳后为小米。

  5. 昔之圣王禹、汤、文、武，兼爱天下之百姓，率以尊天事鬼，其利人多，故天福之，使立为天子，天下诸侯皆宾事之。暴王桀、纣、幽、厉，兼恶天下之百姓，率以诟天侮鬼。其贼人多，故天祸之，使遂失其国家，身死为僇于天下。后世子孙毁之，至今不息。故为不善以得祸者，桀、纣、幽、厉是也。爱人利人以得福者，禹、汤、文、武是也。爱人利人以得福者有矣，恶人贼人以得祸者亦有矣！

  • 笔记 : 本节非常重要，第一次提出了墨家的重要概念“兼爱”。这也是儒墨两家冲突比较深刻的地方。儒家提倡仁爱。仁爱从人际关系出发，因此必须服从于君君臣臣父父子子的规则。而墨家从根本上反对儒家以君为天，以夫为纲的构想。墨家说得很直“天下之为父母者众，而仁者寡”，“天下之为学者众，而仁者寡”，“天下之为君者众，而仁者寡”。这是明着和儒家作对，从根本上就把儒家的人伦治国理念否决掉了。

    墨家为了替换儒家的仁爱，提出了两个概念。一个是“兼爱”。墨家认为君王应该兼爱百姓，“昔之圣王禹、汤、文、武，兼爱天下之百姓，率以尊天事鬼，其利人多，故天福之”。百姓之间也应该相爱，“天必欲人之相爱相利，而不欲人之相恶相贼也。”有没有很熟悉？这就是典型的大同社会的构想，有点像我们的共产主义。

    另一个概念是天。不法于君、不法于学、不法于父，那么法谁呢？墨家的答案是法于天。天这个概念很抽象了。中国有讲究天人合一的传统。在讲天人合一的时候，天是人之外的一个概念，大抵可以理解为自然。中国人相信天和人是会相互呼应的，上天在幽冥之中会指引人，如果人做错了事，上天还会降祸于人。当然天这个概念也是会变的，比如我们在电视剧中就经常听见臣子劝说皇上，要以民为天。这时候，天就是人民。因为对于皇帝来说，他的命运是和人民在一起的。他做了好事，抑或做了坏事，最终是从人民那里反映出来的。所以对于皇帝而言，他的天人合一中的天就是人民。正如人民盼望着自然能带给他们好收成一样，君王也盼望他的人民能够拥护他。

    记得余秋雨分析过儒墨两家的“仁爱”与“兼爱”之辩。他的结论是墨家过于空想而儒家比较实际。在实际的实施上，儒家还是比较可取的。因为天理幽微，比较难循。“爱人利人者，天必福之，恶人贼人者，天必祸之”也不是必然的规律，在现代人眼中，各种资本之恶，合法却不合理的剥削所在多有，但又哪里有人受到天谴呢？因此要以天来约束人，是很难的。比较现实的还是儒家那套基于人伦的做法。

    但墨家思想也并非一无是处。当一门学说垄断历史长达几千年，就会发展过度。中国几千年的历史，一直都是君权，父权不断提升的过程。这种畸形的权力放大了墨子所说的君主，学者，父亲不仁的效应。以至于到了明清时期，出现了封建礼教吃人的现象。然后就是德先生赛先生进入中国，墨子的空想居然就实现了。21世纪最大的信仰就是信奉科学，科学也可以被理解为自然、天理，因此科学的成功也是间接实现了墨子的理想。虽然这么说有点阿Q精神，但毕竟还是实现了嘛，哈哈哈。
