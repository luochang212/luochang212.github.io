author: “Chang Luo” date: 2019-08-22 title: “美国工厂里的劳资关系” categories: [ “Development”, “JavaScript” ] tags: [“Monaco editor”, “web speech API”] weight: 10 description: “Monaco Speech Editor 是一款在线代码编辑器，它能为视障人群提供语音支持，以帮助他们阅读、编辑和编译代码。”

首先，这是一部很好的社会主义题材电影。产业工人和投资方的矛盾在异国背景下被重新审视。

从中国人的视角来看，从前中国人只是想养活自己和家人，现在想去哪里玩就去哪里玩，想买什么就买什么。比以前幸福多了。（概括自影片中王的叙述）而在前司是通用公司的美国工人看来，从前他在通用公司用宽松工作换来体面工资的日子已经一去不复返了。如今他再也挣不到那么丰厚的工资了。

同样的劳碌，一边是日子越过越好，另一边是辉煌不再，这中间心理上的差距自是不可同日而语了。

除了心理，造就两国工人在工作态度上的差异的还有文化。中国工人逆来顺受，主管让排成一队，报数。甚至让工人大喊诸如“对不起”，“请”，这样毫无意义的词汇，工人也是没有丝毫抗拒就喊了。这一幕让我想起《银翼杀手2049》中的片段。片中也出现过类似的词汇，是用来给生化人校对情感用的。然而现实比电影还魔幻，虽不是有意致敬，却能处处暗合。

镜头切换到美国。从中国游历归来，学习了中国管理经验的美国主管，正在尝试将中国主管的经验应用在美国工人身上。但这一开始就遭遇了困难。美国员工甚至不愿意整齐地排成一排。他们眼神轻蔑，站姿随意，对主管的指示爱答不理。最终连主管都觉得尴尬了，只是问了几句关于生产安全的话，就让大家开工去了。

从这里美国人的反应，就能看出他们在心中对自由、独立、尊重都有着极高的需求。管理者试图以军事化手段来管理他们，让他们无条件执行命令，其实就等同于剥夺了他们思考的权利，连带着还侵犯了他们的独立和自由。对工人来说，这当然是无法接受。反观中国人，对这些东西甚至不存有概念，当然也就谈不上侵犯，于是中国人表现出的自然也就只有驯服。

影片还表现了中国管理者对美国工人所谓自由、独立、尊重的嗤之以鼻。这种蔑视自有道理。自由 = 散漫，工会 = 低效，这种公式在客观上也是成立的。中国工厂之所以能打败产业同行，凭借的就是高度的组织性和纪律性。美国工人不愿接受这种新价值观的洗礼，还想靠工会那一套争取自己的利益，那么迟早也会走上通用衰败的老路。

不止管理者明白，连工人自己也明白，或者说害怕了。即使举办了激动人心的游行，唱着劳工团结起来的歌，还是没能阻止大部分工人投出反对票。这也就意味着工人的失败，投资方的胜利。这场失败的意义重大，因为这是文化上的失败，工人们已经失去自信，不再相信自己的力量了。所以这也是投资方彻底的胜利，因为他们不但让产业工人丧失了组建工会的权力，甚至连他们的勇气都一并消灭了。

这就是资本主义的可怕和罪恶。他们为了生产效率，最先消灭的是人的自由、独立和尊重。然后还要消弭人追求自由、独立和尊重的勇气。

然而故事可能还远远没有结束。在可见的未来，随着智能化、自动化的进步，投资方最终连劳动力这个要素都有可能消灭。经济三要素：资本、技术、劳动力，最终资本为王，技术为臣，而劳动力将被机器彻底取代，不再被需要。而这些不再被社会需要的人，今后可能一分钱都拿不到。

如果未来真是这样一条资本和技术媾和之路，那么再不做点什么，我们离赛博朋克真就不远了。

title = “不希望家长用打骂的方式教育孩子” description = “亲子间发生矛盾时，往往没有外人在场。大家都身处其中，谁对谁错其实很难分辨。在无法说清的情况下，家长和孩子双方的心里其实都是很不爽的。” tags = [ “development”, ] date = “2015-05-06” categories = [ “Life”, “Discussion”, ]

亲子间发生矛盾时，往往没有外人在场。大家都身处其中，谁对谁错其实很难分辨。

在无法说清的情况下，家长和孩子双方的心里其实都是很不爽的。

只不过家长与孩子相比，无论是在思维逻辑还是体力上都有压倒性的优势，如果争论或是打起来，一定都是孩子吃亏。兼之家长若是看孩子不顺眼，就容易产生一种“你是老子生的，老子爱怎么打你就怎么打你”的心理。接着就觉得你身体都是我给的，现在居然还来跟我讲道理？你没有这个资格。于是就会自觉出现如下语句：“反正就是你错了”，“你就得给我承认错误”blablabla~

骂是骂爽了，打说不定也打爽了。结果呢？如果果然是孩子错了，那么这一番打骂只会让孩子更加疑惑不解。因为这番“教育”既没有让孩子明白为什么错了，又传递了“谁有力量谁就有道理”这种错觉。人毕竟是有自我认知的动物，如果用条件反射（打骂）这种简单粗暴的方法来教育，只能是家长一厢情愿的认为“我是为你好”。事实上这种教育方式往往是适得其反的。

那么是不是一定不能对孩子使用暴力呢？个人以为要看情况。如果孩子的行为恶劣，甚至触犯了道德底线。那么单单以“惩罚”而非“教育”为目的，施加适当惩戒，让他（她）意识到犯错受罚的道理，以期他（她）能够及时的悬崖勒马。也还是有一定道理的。

不过想想，会产生这样孩子的家庭。其家庭本身或多或少也一定存在着问题。实不期这样家庭中的父母能够通情达理。

如果是家长错了。比如冤枉孩子了。再比如父母由于脾气不好抑或是事业不顺遂，就纯粹地通过打骂孩子，来发泄心中的无名之火。其实也极有可能。这样打骂孩子就更加不对了。

总之，主张言传身教，主张以理服人。

author: “Chang Luo” date: 2024-04-21 title: “极简 DataLoader” categories: [ “DeepLearning” ] tags: [ “Python”, “RNN” ] weight: 10 description: “冬天就是要多做手部运动，让手暖起来”

class MyDataLoader:
    def __init__(self):
        self.i = 0
        self._lst: list

    @property
    def lst(self):
        return self._lst

    @lst.setter
    def lst(self, value):
        self._lst = value

    def __iter__(self):
        return self

    def __next__(self):
        if self.i < len(self._lst):
            i = self.i
            self.i += 1
            return self._lst[i]
        else:
            self.i = 0
            raise StopIteration
