亲子间发生矛盾时，往往没有外人在场。大家都身处其中，谁对谁错其实很难分辨。

在无法说清的情况下，家长和孩子双方的心里其实都是很不爽的。

只不过家长与孩子相比，无论是在思维逻辑还是体力上都有压倒性的优势，如果争论或是打起来，一定都是孩子吃亏。兼之家长若是看孩子不顺眼，就容易产生一种“你是老子生的，老子爱怎么打你就怎么打你”的心理。接着就觉得你身体都是我给的，现在居然还来跟我讲道理？你没有这个资格。于是就会自觉出现如下语句：“反正就是你错了”，“你就得给我承认错误”blablabla~

骂是骂爽了，打说不定也打爽了。结果呢？如果果然是孩子错了，那么这一番打骂只会让孩子更加疑惑不解。因为这番“教育”既没有让孩子明白为什么错了，又传递了“谁有力量谁就有道理”这种错觉。人毕竟是有自我认知的动物，如果用条件反射（打骂）这种简单粗暴的方法来教育，只能是家长一厢情愿的认为“我是为你好”。事实上这种教育方式往往是适得其反的。

那么是不是一定不能对孩子使用暴力呢？个人以为要看情况。如果孩子的行为恶劣，甚至触犯了道德底线。那么单单以“惩罚”而非“教育”为目的，施加适当惩戒，让他（她）意识到犯错受罚的道理，以期他（她）能够及时的悬崖勒马。也还是有一定道理的。

不过想想，会产生这样孩子的家庭。其家庭本身或多或少也一定存在着问题。实不期这样家庭中的父母能够通情达理。

如果是家长错了。比如冤枉孩子了。再比如父母由于脾气不好抑或是事业不顺遂，就纯粹地通过打骂孩子，来发泄心中的无名之火。其实也极有可能。这样打骂孩子就更加不对了。

总之，主张言传身教，主张以理服人。

author: “Chang Luo” date: 2024-04-21 title: “极简 DataLoader” categories: [ “DeepLearning” ] tags: [ “Python”, “RNN” ] weight: 10 description: “冬天就是要多做手部运动，让手暖起来”

class MyDataLoader:
    def __init__(self):
        self.i = 0
        self._lst: list

    @property
    def lst(self):
        return self._lst

    @lst.setter
    def lst(self, value):
        self._lst = value

    def __iter__(self):
        return self

    def __next__(self):
        if self.i < len(self._lst):
            i = self.i
            self.i += 1
            return self._lst[i]
        else:
            self.i = 0
            raise StopIteration

author: “Chang Luo” date: 2023-10-10 title: “什么保险值得买？” categories: [ “insurance”, “analysis” ] tags: [ “python”, “NPV”, “IRR” ] weight: 10 description: “微小的利率，经过时间的累积，可以让本金翻倍，这就是时间的复利。可不要被保险公司蒙蔽了，即使 30 年后返还 本金 * 2，它还有得赚呢”

未来的钱，也许没你想的那么值钱，这是一件挺反直觉的事。要解释这一切，还要从折现率说起

GitHub 项目地址：insurance.ipynb

有位朋友买重疾险，保险顾问给的方案是，身故后返还两倍于累计缴纳金额的钱。听起来像捡便宜，究竟是怎么回事呢？

保险顾问给的具体方案是：

  • 支出：分 30 年支付重疾险，每年付 9000 元，共 27 万元
  • 收益：享受终生重疾险；身故后保险公司一次性返还 50 万元

乍看很划算。是不是真香，算过才知道。

为了把问题讲明白，先引一下背景知识。

一、折现率和净现值

1.1 什么是折现率

由于生息潜力和通胀预期，同样数额的钱在未来更不值钱。将未来的钱折算成现在的钱，存在折扣率，我们叫它 折现率 (Discount rate)。举个例子，如果你认为一年后的 107 元与现在的 100 元相当，折现率就是 7%。

1.2 什么是现值

按折现率，把“未来的钱”折算成“现在的钱”的值就是 现值 (Present value)。

$一年后的107元的现值 = \frac{107}{1 + 折现率} = \frac{107}{1 + 7\%} = 100$
