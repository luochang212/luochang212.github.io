// 01 项目概述 overview

本科期间爬取的个人微博数据在硬盘里沉寂了两年，趁机会用 Pandas + seaborn 重新挖掘，从多个维度可视化呈现社交媒体行为模式。

数据经 Pandas 清洗处理后，通过 seaborn 与 matplotlib 生成统计图表，完整分析流程在 Jupyter Notebook 中呈现。

// 02 分析维度 dimensions

  • >时间维度
    发博频率的时间序列分析，按小时 / 天 / 月聚合
  • >地理维度
    发博位置热力图与地理分布可视化
  • >来源维度
    微博发布设备 / 客户端来源统计
  • >内容维度
    文本长度分布、词云与关键词频率分析
  • >互动维度
    转发、评论、点赞等互动数据统计
