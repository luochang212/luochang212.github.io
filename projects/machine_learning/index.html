// 01 关于此项目 about

本项目使用主成分分析（PCA）对 MNIST 手写数字数据集进行特征降维。原始 784 维像素空间被压缩至低维主成分空间，在保留主要判别信息的同时大幅降低计算复杂度。

降维后训练分类器，实现高效、准确的手写数字识别。完整代码与实验记录托管于 GitHub。

// 02 技术要点 tech_specs

  • >MNIST 数据集
    70,000 张 28×28 灰度手写数字图像
  • >PCA 降维
    从 784 维降至 k 维，保留 95% 方差信息
  • >分类器
    基于降维特征训练，对比不同 k 值的分类精度
  • >可视化
    主成分空间中的类别分布、重构误差分析
