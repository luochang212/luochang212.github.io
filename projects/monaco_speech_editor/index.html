// 01 项目背景 background

编程领域对视障人群的支持长期不足。本项目在 VS Code 同款内核 Monaco Editor 上集成语音辅助功能，让视障用户也能通过键盘导航和语音反馈高效编程。

核心创新在于针对编码场景设计了四种差异化语音模式，覆盖阅读、调试、浏览等不同需求阶段。

// 02 功能特性 features

  • >鼠标悬浮提示
    光标 hover 时语音读出变量名、类型等信息
  • >快捷键操作
    纯键盘操控，无需鼠标即可完成代码导航与编辑
  • >字符模式
    逐字符朗读，适用于 debug 场景
  • >代码模式
    按语法结构朗读，适用于代码浏览
  • >音乐模式 / 全局模式
    非语音音频反馈与全局概览，适配不同阅读需求
