// 01 项目介绍 about

《LangGraph 1.0 完全指南》是一套面向 Agent 开发者的开源教程，帮助读者同时掌握 LangGraph 与 LangChain 的核心能力。

教程从构建第一个 ReAct Agent 开始，通过可运行的 Notebook 讲解状态、节点、边与持久化等基础概念，再逐步扩展到中间件、记忆、MCP、RAG、多智能体和应用开发。

// 02 内容结构 curriculum

  • >基础框架
    ReAct Agent、StateGraph、中间件与人机交互
  • >状态与上下文
    短期记忆、长期记忆、Store、Runtime 与上下文工程
  • >工作流扩展
    MCP Server、监督者模式、并行处理、RAG 与网络搜索
  • >应用实践
    Deep Agents、Gradio 流式对话应用与 langgraph-cli 调试

// 03 实践章节 hands_on

第 13 章使用 Gradio 与 LangChain 构建流式对话智能体，展示工具调用、过程反馈、数据可视化与对话界面的组合方式。第 14 章介绍如何使用 langgraph dev 启动本地调试页面。

// 04 开源协作 open_source

教程源码、Notebook 与构建配置均托管在 GitHub。你可以在线阅读完整电子书，也可以提交 Issue 或 Pull Request，参与修正示例、补充说明与完善章节内容。
